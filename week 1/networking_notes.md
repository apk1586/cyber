# Networking Fundamentals

## OSI Model (7 Layers)
7. **Application:** User interface (HTTP, FTP, SMTP)
6. **Presentation:** Data formatting, encryption (SSL/TLS)
5. **Session:** Establishing, maintaining, and terminating sessions
4. **Transport:** End-to-end connections and reliability (TCP, UDP)
3. **Network:** Path determination and logical addressing (IP)
2. **Data Link:** Physical addressing (MAC)
1. **Physical:** Media, signal, and binary transmission

## TCP/IP Model (4 Layers)
1. Application
2. Transport
3. Internet
4. Network Access

## Key Protocols
- **DNS (Domain Name System):** Resolves human-readable domains to IP addresses (Port 53).
- **HTTP/HTTPS:** Web traffic (Port 80/443).
- **SSH (Secure Shell):** Secure remote administration (Port 22).
